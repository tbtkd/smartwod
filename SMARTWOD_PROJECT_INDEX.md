
# SMARTWOD_PROJECT_INDEX.md
Índice Maestro de Documentación — SMARTWOD

Este documento funciona como **punto central de navegación** para toda la documentación del proyecto.
Su objetivo es que cualquier desarrollador (o tú mismo en el futuro) pueda entender rápidamente
la estructura, arquitectura y evolución del sistema.

Se recomienda mantener este archivo **siempre actualizado** cuando se agregue nueva documentación.

# Cómo usar este archivo

1. Este archivo debe ubicarse en la **raíz del repositorio**.
2. Todos los documentos importantes deben estar listados aquí.
3. Cuando agregues un nuevo documento `.md`, debes registrarlo en este índice.
4. Este archivo sirve como **mapa de navegación del proyecto**.
