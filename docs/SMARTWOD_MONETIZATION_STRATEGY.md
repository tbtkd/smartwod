
# MONETIZATION_STRATEGY.md
SMARTWOD — Estrategia de Monetización

Este documento describe posibles estrategias de monetización para la aplicación.

## Modelo Freemium

Versión gratuita incluye:

- AMRAP
- EMOM
- TABATA
- FOR TIME
- historial básico
- notas

Versión premium incluye:

- estadísticas avanzadas
- filtros de historial
- dashboard de progreso

## Estadísticas Premium

Ejemplos:

Progreso semanal
Tiempo entrenado por semana
Distribución por tipo de entrenamiento

## Dashboard personal

Total workouts
Tiempo entrenado
Promedio por sesión

## Precio sugerido

Compra única:

3.99 USD

Alternativa:

suscripción anual de bajo costo.

## Filosofía

Las funciones principales siempre deben permanecer gratuitas.
El premium debe mejorar la experiencia sin bloquear el uso básico.
